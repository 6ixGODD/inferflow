# Instance Segmentation Example

Demonstrates YOLOv5 instance segmentation with mask overlay.

## Features

- YOLOv5-Seg pretrained model
- Instance segmentation masks
- Colored mask overlay
- Bounding boxes
- Per-instance statistics

## Usage

```bash
python segment.py
```

## Output

* `output/segmented.jpg` - Image with segmentation masks

## Model

* Architecture: YOLOv5s-Seg
* Source: Ultralytics
* Classes: COCO dataset (80 classes)
