# Object Detection Example

Demonstrates YOLOv5 object detection with bounding box visualization.

## Features

- YOLOv5s pretrained model
- Multi-object detection
- Bounding box drawing
- Confidence scores
- Class labels

## Usage

```bash
python detect.py
```

## Output

* `output/detected.jpg` - Image with bounding boxes

## Model

* Architecture: YOLOv5s
* Source: Ultralytics
* Classes: COCO dataset (80 classes)
